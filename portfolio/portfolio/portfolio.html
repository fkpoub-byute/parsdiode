طراحی مدارهای مجتمع با استفاده از نرم‌افزارهای حرفه‌ای

تاریخ انجام: 1403/03/20
مدت زمان: 3 ماه
الکترونیک
طراحی PCB Altium Designer سیمولیشن تست عملی

توضیحات پروژه

این پروژه شامل طراحی و پیاده‌سازی یک مدار مجتمع پیشرفته برای کاربردهای صنعتی بود. مدار طراحی شده قادر به پردازش سیگنال‌های آنالوگ و دیجیتال با دقت بالا می‌باشد.

در این پروژه از نرم‌افزارهای حرفه‌ای Altium Designer برای طراحی PCB و نرم‌افزارهای شبیه‌سازی مانند Proteus و Multisim برای تحلیل عملکرد مدار استفاده شد.

مشخصات فنی

  • ولتاژ کاری: 3.3V DC
  • فرکانس کاری: 100MHz
  • تعداد لایه‌ها: 4 لایه
  • ابعاد PCB: 80mm × 60mm
  • تعداد قطعات: 127 عدد

جزئیات فنی

نرم‌افزارهای استفاده شده

  • Altium Designer (طراحی PCB)
  • Proteus (شبیه‌سازی)
  • Multisim (تحلیل مدار)
  • AutoCAD (طراحی مکانیکی)

قطعات اصلی

  • میکروکنترلر ARM Cortex-M4
  • ADC 16-bit با نرخ نمونه‌برداری 1MSPS
  • حافظه DDR3 512MB
  • رابط USB 3.0
